# Atmosphère — Weather Dashboard

A clean, real-time weather dashboard built with vanilla HTML, CSS, and JavaScript. No frameworks. Powered by the Open-Meteo API.

## Features

- 🔍 Search any city worldwide
- 🌡️ Current conditions: temperature, feels-like, humidity, pressure, visibility, UV index, wind
- ⏰ 24-hour forecast with precipitation probability
- 📅 7-day daily forecast
- 🌅 Sunrise / sunset times
- 🎨 Dynamic theme that adapts to weather conditions (sunny, rainy, snowy, thunderstorm, night)
- ♿ Respects `prefers-reduced-motion`

## Tech Stack

- HTML5, CSS3, vanilla JavaScript (ES6+)
- [Open-Meteo API](https://open-meteo.com/) for weather and geocoding
- Google Fonts: Cormorant Garamond + Space Mono

## Run Locally

Just open `index.html` in a browser, or serve it locally:

\`\`\`bash
python3 -m http.server 5500
\`\`\`

Then visit `http://127.0.0.1:5500`
